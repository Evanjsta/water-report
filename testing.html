The Shearwater Pool Report

The water is tested using a Lamotte Spin Touch, the same kind of water testing device used professional pool stores. Testing is done following all manufacturer's instructions to make sure that the best possible results are recorded. The Lamotte Spin Touch uses a series of reagents that are added to the water sample. The reagents react with the chemicals in the water, and the device measures the resulting color changes to determine the concentration of various substances in the water using a laser.
